Projetos

Todos os projetos que desenvolvi foram realizados durante o curso técnico em Desenvolvimento de Sistemas, entre 2023 e 2024. Durante esse período, trabalhei tanto em equipe quanto individualmente, mas destaco meu trabalho em grupo, entregue para a conclusão do curso. Quando trabalho em grupo, tenho o costume de ajudar meus colegas a resolver os problemas deles, nesse sentido sou bastante proativo, minha parte favorita de um trabalho é resolver problemas.

As tecnologias e ferramentas principais utilizadas no projeto foram: Node.js, Handlebars, SQL, MySql Workbench, VS Code e Figma. Link GitHub
